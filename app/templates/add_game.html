How to Find the BGG ID

To add a new game, you can use the BoardGameGeek ID (BGG ID) for accuracy. Follow these steps:

  1. Go to BoardGameGeek and search for your game.

  2. Open the game's page and locate the BGG ID in the URL. The ID is the number that appears in the link.

    For example, in this URL:

    https:​//boardgamegeek.com​/boardgame​/342942​/ark-nova

    The BGG ID is 342942.

If you cannot find the BGG ID, you can enter the game name manually. Providing the BGG ID ensures more accurate game details.
